5-STEP PROCESS FROM INPUT TO PROFESSIONAL REPORTS
================================================================================

STEP 1: PARSE PDF & EXTRACT 125 QUESTIONS
─────────────────────────────────────────
INPUT:  exam_1_week1.pdf
        ├─ Page 1-10: Questions 1-30
        ├─ Page 11-20: Questions 31-60
        ├─ Page 21-30: Questions 61-90
        └─ Page 31-35: Questions 91-125

PROCESS:
  1. Load PDF file
  2. Extract text from all pages
  3. Identify question boundaries (Q1, Q2, ... Q125)
  4. Parse question text and 4 options (A, B, C, D)
  5. Create structured data for each question

OUTPUT: Structured Questions
  {
    "1": {
      "text": "What is AES?",
      "options": {"A": "...", "B": "...", "C": "...", "D": "..."}
    },
    "2": { ... },
    ...
    "125": { ... }
  }

TIME: ~2 seconds


STEP 2: PARSE EXCEL & EXTRACT STUDENT ANSWERS
──────────────────────────────────────────────
INPUT:  exam_1_batch_dec25.xlsx

        Column A    Column B      Column C    Column D      Column E
        Question    Alice         Bob         Carol         David
        ──────────────────────────────────────────────────────────
        Q1          A             B           A             C
        Q2          B             C           B             D
        Q3          C             A           D             A
        ...
        Q125        A             B           C             D

PROCESS:
  1. Load Excel file
  2. Read column headers (student names)
  3. Extract answers for each student
  4. Normalize formats (handle A, 1-A, A,B,C,D, etc.)
  5. Map Q1-Q125 for each student

OUTPUT: Student Answers
  {
    "Alice": {
      "1": "A", "2": "B", "3": "C", ..., "125": "A"
    },
    "Bob": {
      "1": "B", "2": "C", "3": "A", ..., "125": "B"
    },
    ...
  }

SUPPORTED FORMATS (Auto-Normalized):
  ✓ A (single letter)
  ✓ 1-A (with dash)
  ✓ 1-A,2-B,3-C,4-D (multi-part with dashes)
  ✓ 1A2B3C4D (no separators)
  ✓ A,B,C,D (positional)
  ✓ a,b,c,d (lowercase)

TIME: ~1 second


STEP 3: MATCH ANSWERS TO ANSWER KEY & CALCULATE SCORES
───────────────────────────────────────────────────────
INPUT:
  • Student Answers (from Step 2)
  • Answer Key: exam_1_week1_answer_key.json

ANSWER KEY: {"1": {"letter": "A", ...}, "2": {"letter": "B", ...}, ... }

PROCESS:
  For each student:
    For each question (1-125):
      if student_answer == answer_key_answer:
        mark CORRECT ✓
      else:
        mark WRONG ✗
        track question number

SCORING EXAMPLE (Alice):
  Correct: 98 out of 125
  Wrong: 27 out of 125
  Score: 98/125 = 78.4%
  Wrong Questions: [5, 12, 23, 45, ...]

OUTPUT: Scored Results
  {
    "Alice": {
      "correct": 98,
      "wrong": 27,
      "percentage": 78.4,
      "wrong_questions": [5, 12, 23, ...]
    },
    "Bob": {
      "correct": 105,

      "percentage": 84.0,
      "wrong_questions": [8, 19, 34, ...]
    },
    ...
  }
